Estrutura básica de uma arquivo HTML

Um arquivo HTML possui uma estrutura básica necessária ao seu perfeito funcionamento.

Esta estrutura é formada por informações sobre o arquivo e duas partes principais: head e body

Cada parte que compõem um arquivo HTML é chamada de tag, que são trechos de código escritos entre os caracteres < e >

As tags de abertura são representadas das seguinte forma:

<nome_da_tag>

E as tags de fechamento devem incluio o caractere /:

</nome_da_tag>

Indicando o tipo de documento

No início dos documentos HTML é necessário incluir a declaração de tipo de documento, para isso, utilizamos a tag DOCTYPE

<!DOCTYPE html>

Note que a tag <!DOCTYPE html> é uma tag que não possui uma tag de fechamento, por isso também é conhecida como self-closed.

A tag <html>

Após a declaração do tipo de documento, é necessário incluir a tag <html> em nosso documento. É nessa tag que informamos qual é o idioma e a localização geográfica do nosso site:

<html lang="pt-BR">

Note que dentro do atributo lang temos um valor composto por dois termos, pt e BR, onde pt corresponde ao Código de Idioma (Language Codes) definidos pela ISO 639-1. A lista completa de Language Codes pode ser vista no link https://www.w3schools.com/tags/ref_language_codes.asp

Já o BR corresponde ao Código do País (Country Code), que também obedece a um padrão ISO. A lista completa de Country Codes pode acessada no link https://www.w3schools.com/tags/ref_country_codes.asp

A tag <head>

A tag <head> é responsável por conter informações meta sobre o documento HTML. Diferente da tag <body>, que exibe conteúdo visível para o usuário, tudo o que está dentro do <head> é voltado para configurações, descrições, links para recursos externos e outros dados que ajudam navegadores e motores de busca a interpretar e apresentar corretamente a página.

Um exemplo simples de uso da tag <head> seria:

<head>
    <meta charset="UTF-8">
    <meta name="viewport" content="width=device-width, initial-scale=1.0">
    <title>Document</title>
</head>

<meta charset="UTF-8">

Esse elemento define a codificação de caracteres do documento, e o valor UTF-8 é atualmente o mais recomendado, pois suporta praticamente todos os caracteres e símbolos usados mundialmente — incluindo acentos e caracteres especiais do português.

<meta name="viewport" content="width=device-width, initial-scale=1.0">

Essa configuração é essencial para garantir que o site seja exibido corretamente em dispositivos móveis. O atributo width=device-width ajusta a largura da página para a largura da tela do dispositivo, enquanto initial-scale=1.0 define o nível inicial de zoom. Sem essa meta tag, o layout pode ficar desproporcional em celulares e tablets.

<title>Document</title>

A tag <title> define o título da página, que aparece na aba do navegador e também é usado como título principal pelo Google e outros motores de busca nos resultados de pesquisa. É recomendado usar um título descritivo e relevante para melhorar a usabilidade e o SEO (Search Engine Optimization).

O conteúdo dentro de <head> não é visível diretamente para o usuário, mas é fundamental para o funcionamento adequado do site, sua acessibilidade, responsividade e posicionamento nos mecanismos de busca.
