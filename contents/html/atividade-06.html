HTML - Atividade 06

Trabalhando com Mídias

Crie uma pasta chamada atividade-06 e dentro dela, crie os seguintes arquivos:

  • index.html
  • audio.html
  • video.html

Crie, dentro da pasta atividade-06 a pasta assets, que é onde devem ser aramazenados os arquivos de mídia.

Página index.html

A página index.html deve conter:

  • Um título <h1> com o texto Exercícios de Mídias em HTML.
  • Uma estrutura de navegação para acessar as outras páginas (audio.html e video.html).
  • Um pequeno parágrafo explicando que esta atividade é destinada ao aprendizado das tags <audio> e <video>.

Página audio.html

A página audio.html deve conter:

  • Um título da página.
  • Um exemplo simples de áudio com o atributo controls.
  • Um exemplo de áudio com múltiplos formatos (.mp3,.wav e .ogg) utilizando a tag <source>.
  • Um exemplo de áudio utilizando loop e muted.
  • Um caso de áudio com caminho errado para demonstrar a mensagem alternativa dentro da tag.
  • Estrutura de navegação para as demais páginas.

Página video.html

A página video.html deve conter:

  • Um título da página.
  • Um vídeo simples com controls e largura definida (width).
  • Um vídeo com múltiplos formatos (.mp4, .webm) utilizando a tag <source>.
  • Um vídeo com os atributos autoplay, loop, muted e poster (imagem de capa).
  • Um caso de vídeo com caminho errado para demonstrar a mensagem alternativa dentro da tag.
  • Estrutura de navegação para as demais páginas.

Resumo da Atividade

- A página index.html serve como introdução e hub de navegação.
- A página audio.html pratica o uso da tag <audio> com diferentes atributos e formatos.
- A página video.html trabalha a tag <video>, formatos alternativos e recursos adicionais como poster, autoplay e loop.
