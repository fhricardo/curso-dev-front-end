Atividade 08 - Estrutura de Site com Tags Semânticas

Nesta atividade você deverá criar a estrutura de um site simples com 4 páginas, utilizando corretamente as principais tags semânticas do HTML5. O objetivo é praticar a organização de conteúdo e a construção de layouts básicos com significado claro.

Regras da Atividade

  • O site deve ter 4 páginas: index.html, sobre.html, servicos.html e contato.html.
  • Cada página deve conter:
    • <header> com o título do site e um menu de navegação (links para todas as páginas).
    • <main> com o conteúdo principal daquela página.
    • <aside> com um conteúdo complementar (pode ser uma lista de links, propaganda ou curiosidade).
    • <footer> com direitos autorais e informações de rodapé.
  • Utilize a tag <nav> dentro do <header> para criar o menu de navegação.
  • Use a tag <section> ou <article> quando for necessário dividir o conteúdo principal.
  • Não é necessário aplicar CSS nesta atividade (apenas HTML).

Exemplo de Estrutura da Página Inicial

<!DOCTYPE html>
<html lang="pt-br">
<head>
    <meta charset="UTF-8">
    <title>Meu Site</title>
</head>
<body>

    <header>
        <h1>Meu Site</h1>
        <nav>
            <a href="index.html">Início</a> |
            <a href="sobre.html">Sobre</a> |
            <a href="servicos.html">Serviços</a> |
            <a href="contato.html">Contato</a>
        </nav>
    </header>

    <main>
        <section>
            <h2>Bem-vindo ao site!</h2>
            <p>Este é o conteúdo principal da página inicial.</p>
        </section>
    </main>

    <aside>
        <h3>Links úteis</h3>
        <ul>
            <li><a href="#">Link 1</a></li>
            <li><a href="#">Link 2</a></li>
        </ul>
    </aside>

    <footer>
        <p>&copy; 2025 - Meu Site. Todos os direitos reservados.</p>
    </footer>

</body>
</html>

Desafio Extra

Personalize o conteúdo de cada página (Sobre, Serviços e Contato), utilizando sempre as tags semânticas adequadas para dar significado ao conteúdo. Ao final, você terá um site de múltiplas páginas totalmente estruturado em HTML5.
