Links e Navegação

A tag anchor <a>

A tag <a> é usada para criar links, permitindo a navegação entre páginas ou para diferentes partes de um mesmo documento. É um dos elementos mais importantes da web, pois conecta conteúdos e transforma páginas isoladas em uma rede interligada.

O atributo mais comum é o href, que define o destino do link. Esse destino pode ser uma URL externa, uma página interna do mesmo site ou até mesmo um ponto específico dentro da mesma página.

Além disso, podemos usar o atributo target para controlar como o link será aberto. Por exemplo: _self abre na mesma aba (padrão) e _blank abre em uma nova aba do navegador.

Barras de Navegação

Em HTML, uma barra de navegação é geralmente criada usando a tag <nav>, que representa uma seção da página destinada a links de navegação. Dentro dela, utilizamos uma lista não ordenada (<ul>) para organizar os itens, e cada item da lista contém um link (<a>).

Mesmo sem CSS, essa estrutura já é compreensível por navegadores, mecanismos de busca e leitores de tela, garantindo acessibilidade e semântica adequada. 

<nav>
    <ul>
        <li><a href="index.html">Início</a></li>
        <li><a href="sobre.html">Sobre</a></li>
        <li><a href="servicos.html">Serviços</a></li>
        <li><a href="contato.html">Contato</a></li>
    </ul>
</nav>
  • Início
  • Sobre
  • Serviços
  • Contato

Barras de Navegação em uma Página

Não existe um limite fixo para a quantidade de barras de navegação em uma página, mas é importante manter a organização e a clareza para que o usuário não se perca. Normalmente, encontramos de 1 a 3 tipos de navegação em um site, dependendo da complexidade do conteúdo.

Tipos mais comuns de navegação

1. Navegação Principal (ou Global)

É a barra principal do site, geralmente localizada no topo do cabeçalho. Nela ficam os links mais importantes, como: Início, Sobre, Serviços e Contato. 

<header>
    <nav class="principal">
        <ul>
            <li><a href="#">Início</a></li>
            <li><a href="#">Sobre</a></li>
            <li><a href="#">Serviços</a></li>
            <li><a href="#">Contato</a></li>
        </ul>
    </nav>
</header>
Início | Sobre | Serviços | Contato

2. Navegação Secundária

Fica abaixo ou ao lado da navegação principal e mostra subcategorias ou conteúdos relacionados. 

<nav class="secundaria">
    <ul>
        <li><a href="#">Masculino</a></li>
        <li><a href="#">Feminino</a></li>
        <li><a href="#">Infantil</a></li>
    </ul>
</nav>
Masculino | Feminino | Infantil

3. Navegação de Rodapé

Localizada no final da página, geralmente repete os links principais e inclui links extras como Política de Privacidade e Termos de Uso. 

<footer>
    <nav class="rodape">
        <ul>
            <li><a href="#">Início </a></li>
            <li><a href="#"> Política de Privacidade </a></li>
            <li><a href="#"> Termos de Uso</a></li>
        </ul>
    </nav>
</footer>
Início | Política de Privacidade | Termos de Uso
