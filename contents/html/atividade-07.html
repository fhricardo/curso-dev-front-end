HTML - Atividade 07

Trabalhando com Formulários

Crie uma pasta chamada atividade-07 e dentro dela, crie os seguintes arquivos:

  • index.html
  • cadastro.html
  • contato.html

Página index.html

A página index.html deve conter:

  • Um título <h1> com o texto Exercícios de Formulários em HTML.
  • Uma estrutura de navegação para acessar as páginas cadastro.html e contato.html.
  • Um pequeno parágrafo explicando que esta atividade é destinada ao aprendizado de formulários em HTML.

Página cadastro.html

A página cadastro.html deve conter:

  • Um título da página.
  • Um formulário de cadastro de usuário contendo os seguintes campos:
    • Nome (texto)
    • E-mail (email)
    • Senha (password)
    • Data de nascimento (date)
    • Gênero (radio: Masculino, Feminino, Outro)
    • Interesses (checkbox: HTML, CSS, JavaScript)
    • Um campo de seleção (select) para escolher um curso (Front-End, Back-End, Design)
  • Botões de submit e reset.
  • Uma estrutura de navegação para as demais páginas.

Página contato.html

A página contato.html deve conter:

  • Um título da página.
  • Um formulário de contato simples com os seguintes campos:
    • Nome (texto)
    • E-mail (email)
    • Assunto (select com pelo menos 3 opções)
    • Mensagem (textarea)
  • Um botão de envio (submit).
  • Estrutura de navegação para as demais páginas.

Resumo da Atividade

- A página index.html funciona como hub de navegação e introdução da atividade.
- A página cadastro.html aplica campos de diferentes tipos (text, password, date, radio, checkbox, select).
- A página contato.html pratica o uso de formulários para comunicação, usando textarea e select.
