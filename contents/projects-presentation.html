Projeto: Apresentação Pessoal

Chegou a hora de aplicar tudo o que aprendemos até agora em um projeto prático! Vamos criar uma landing page de apresentação pessoal. A ideia é montar uma página simples, mas que já tenha aparência de site real, utilizando apenas HTML e CSS com os conteúdos que vimos até aqui.

Objetivo

Criar uma página única com informações básicas de apresentação, organizada em seções, usando textos, cores, fontes, fundos e espaçamentos para dar estilo.

Estrutura do Projeto

A página deve conter pelo menos as seguintes seções:

  • Cabeçalho: título principal do site (seu nome ou nome fictício) e um subtítulo.
  • Seção Sobre: um parágrafo apresentando você (ou o personagem fictício).
  • Seção Interesses/Serviços: lista de hobbies, áreas de interesse ou serviços (pode ser uma lista simples ou blocos com cor de fundo).
  • Seção Contato: texto com e-mail, telefone ou link fictício.
  • Rodapé: um texto simples como "© 2025 Seu Nome – Todos os direitos reservados".

Requisitos de Estilo (CSS)

Você deve aplicar pelo menos:

  • Uma cor de fundo diferente para o cabeçalho.
  • Uso de font-family personalizada (pode usar Google Fonts).
  • Tamanhos diferentes de fonte (font-size) e espessura (font-weight).
  • Pelo menos um exemplo de background-image (pode ser no cabeçalho ou em uma seção).
  • Uso de margin e padding para dar espaçamento entre as seções.
  • Caixas com border ou border-radius para destacar informações.

Exemplo de Estrutura HTML

<header>
    <h1>João da Silva</h1>
    <p>Desenvolvedor Front-end Iniciante</p>
</header>

<section>
    <h2>Sobre Mim</h2>
    <p>Sou apaixonado por tecnologia e estou aprendendo HTML e CSS.</p>
</section>

<section>
    <h2>Meus Interesses</h2>
    <ul>
        <li>Design Web</li>
        <li>Programação</li>
        <li>Música</li>
    </ul>
</section>

<section>
    <h2>Contato</h2>
    <p>E-mail: joao@email.com</p>
</section>

<footer>
    <p>&copy; 2025 João da Silva – Todos os direitos reservados.</p>
</footer>

Resumo

Este projeto vai ajudar você a consolidar o que já aprendeu em HTML e CSS:
- Criar seções organizadas no HTML.
- Usar cores e imagens de fundo.
- Aplicar fontes personalizadas com Google Fonts.
- Trabalhar com tamanhos de texto, margens, paddings e bordas.
- Dar um ar mais profissional a uma página simples.
