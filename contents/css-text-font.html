Propriedades de Fontes e Textos

Além das cores, podemos controlar de forma detalhada como o texto aparece na tela: tipo da fonte, tamanho, espaçamento, estilos e até aplicar fontes externas como as do Google Fonts.

Font Family

A propriedade font-family permite que seja definida a família tipográfica a ser utilizada no projeto. Basicamente, é a propriedade que possibilita escolher a fonte que queremos utilizar em uma determinada parte da nossa página.

p {
    font-family: Arial, Helvetica, sans-serif;
}

Texto com fonte Arial (ou Helvetica/Sans-serif se não disponível).

Web Safe Font

As Web Safe Fonts ou Fontes Seguras para Web são conjuntos de famílias tipográficas comuns na maioria dos dispositivos e que permitem a renderização mais semelhante ao projeto original, reduzindo problemas de layout ocasionados por troca de fontes do projeto

Entendendo as Fontes para Web

Fontes são programas de computadores e, desta forma, precisam estar instaladas para que sejam utilizadas.

Desenvolver um projeto web com uma fonte instalada em nosso computador pode não ser uma boa ideia, uma vez que, após publicado na web, o nosso site não vai contar com aquela fonte utilizada e não temos como identificar se o usuário tem ou não aquela fonte específica instalada, o que pode ocasionar problemas com o layout e a correta exibição do

O bloco de declaração de família tipográfica que utiliza Web Safe Fonts geralmente é apresentado da maneira exibida abaixo:

p {
    font-family: Arial, Helvetica, sans-serif;
}

Onde são apresentadas duas ou mais famílias tipográficas semelhantes, basicamente funcionando da seguinte forma: Utilize a fonte Arial, caso não esteja disponível, utilize Helvetica e, em último caso, utilize sans-serif.

Antes de publicar seu site, verifique como as fontes são exibidas em diferente navegadores e dispositivos.

Font Size e Font Weight

As propriedades font-size e font-weight são responsáveis, respectivamente, pela definição de tamanho e peso (espessura) das fontes no projeto.

p {
    font-size: 20px;     /* tamanho do texto */
    font-weight: bold;   /* espessura: normal, bold, 100–900 */
}

Texto maior e em negrito.

Para utilizar diferentes espessuras em fontes é recomendado utilizar uma família tipográfica que possua as variações desejadas.

Font Style e Text Transform

A propriedade font-style é responsável por incluir o aspecto itálico ou oblíquo em fontes e assim como informado na dica do tópico anterior, é recomendado que a família tipográfica selecionada possua essas variações.

Já a propriedade text-transform permite que a caixa de um conjunto selecionado seja modificada, contando com variações como UPPERCASE (todas as letras em maiúsculas), lowercase (todas as letras em minúsculas) e Capitalize (primeira letra de cada palavra em maiúscula). 

p {
    font-style: italic;       /* normal, italic, oblique */
    text-transform: uppercase; /* lowercase, capitalize */
}

Texto itálico e em MAIÚSCULAS.

Text Decoration

A propriedade text-decoration permite que seja definido o tipo de decoração do texto, por exemplo, em uma tag <a>, por padrão, o texto é exibido sublinhado (underline) e, utilizando esta propriedade, é possível alterar essa exibição.

a {
    text-decoration: none; /* underline, overline, line-through */
}

Line Height e Letter Spacing

O uso da propriedade line-height é essencial para melhorar a legibilidade de textos em páginas web. Ao ajustar a altura das linhas, é possível criar um espaçamento mais confortável entre elas, reduzindo a sensação de "amontoamento" do conteúdo. Isso é especialmente importante em textos longos, pois proporciona uma experiência de leitura mais agradável e menos cansativa para o usuário, além de contribuir para um layout mais organizado e visualmente equilibrado.

Já a propriedade letter-spacing permite controlar o espaçamento entre as letras, o que pode ser útil para destacar títulos, criar efeitos visuais ou melhorar a legibilidade de fontes específicas. Um espaçamento maior entre as letras pode ajudar a evitar que os caracteres pareçam muito próximos, especialmente em tamanhos pequenos ou em fontes mais decorativas. Por outro lado, ajustes sutis podem ser usados para refinar a estética do texto, garantindo harmonia com o design geral da página.

p {
    line-height: 1.8;     /* altura da linha */
    letter-spacing: 3px;  /* espaço entre letras */
}

Texto com maior altura de linha, espaçamento adicional entre letras, proporcionando melhor legibilidade e uma apresentação mais agradável para leitura em diferentes dispositivos.

Google Fonts

Podemos usar fontes externas no site. O Google Fonts oferece diversas opções gratuitas. Basta incluir o link da fonte no HTML e usar no CSS.

  1. Acesse o site do Google Fonts.
  2. Pesquise e selecione a fonte desejada.
  3. Escolha os estilos (weights) da fonte que deseja utilizar.
  4. Copie o link gerado na seção "Use on the web".
  5. Inclua o link no <head> do seu arquivo HTML: 
    <link href="https://fonts.googleapis.com/css2?family=Roboto&display=swap" rel="stylesheet">
  6. No arquivo CSS, utilize a fonte com a propriedade font-family: 
    p {
    font-family: 'Roboto', sans-serif;
}

    7. Atualize a página no navegador para visualizar a nova fonte aplicada.

    Alternativa

  1. Como alternativa, podemos utilizar também o link de importação da fonte diretamente para nosso arquivo CSS.Basta seguir os passo acima, porém ao invés de copiar o link para o arquivo HTML, utilizaremos a regra de importação para o arquivo CSS: 
    2. @import url('https://fonts.googleapis.com/css2?family=Inter:ital,opsz,wght@0,14..32,100..900;1,14..32,100..900&display=swap');
  2. após copiar, basta colar na primeira linha do nosso arquivo CSS e utilizar a propriedade font-family 
    3. p {
    font-family: 'Roboto', sans-serif;
}

Texto usando a fonte Roboto do Google Fonts.

Resumo

- font-family: define a fonte (usar famílias de fallback).
- font-size: controla o tamanho do texto.
- font-weight: espessura da fonte (normal, bold, 100–900).
- font-style: estilo (normal, italic, oblique).
- text-transform: transformação de maiúsculas/minúsculas.
- text-decoration: sublinhado, tachado, etc.
- line-height: altura da linha (importante para legibilidade).
- letter-spacing, word-spacing: espaçamentos de texto.
- Google Fonts: permite usar fontes externas facilmente.
