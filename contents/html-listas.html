Listas em HTML

Em HTML, listas são estruturas usadas para organizar informações em sequência ou em tópicos. Elas são úteis para menus, conteúdos ordenados, glossários, entre outros. Existem três tipos principais de listas: não ordenadas, ordenadas e de definição.

1. Lista Não Ordenada (<ul>)

A lista não ordenada exibe os itens com marcadores simples (como pontos). É indicada quando a ordem dos elementos não importa, por exemplo, uma lista de compras. 

<ul>
    <li>Maçãs</li>
    <li>Bananas</li>
    <li>Laranjas</li>
</ul>
  • Maçãs
  • Bananas
  • Laranjas

2. Lista Ordenada (<ol>)

A lista ordenada mostra os itens numerados (1, 2, 3...) ou em outras formas de contagem (letras ou algarismos romanos, por exemplo). É usada quando a ordem é relevante, como em um passo a passo de instruções. 

<ol>
    <li>Pré-aquecer o forno</li>
    <li>Misturar os ingredientes</li>
    <li>Assar por 30 minutos</li>
</ol>
  1. Pré-aquecer o forno
  2. Misturar os ingredientes
  3. Assar por 30 minutos

3. Lista de Definição (<dl>)

A lista de definição é composta por termos (<dt>) e suas descrições (<dd>). É indicada para glossários, FAQs ou informações onde há um par termo/explicação. 

<dl>
    <dt>HTML</dt>
    <dd>Linguagem de marcação para estruturar páginas web.</dd>

    <dt>CSS</dt>
    <dd>Folhas de estilo em cascata usadas para design e layout.</dd>

    <dt>JavaScript</dt>
    <dd>Linguagem de programação usada para interatividade.</dd>
</dl>
HTML
Linguagem de marcação para estruturar páginas web.
CSS
Folhas de estilo em cascata usadas para design e layout.
JavaScript
Linguagem de programação usada para interatividade.

Resumo

- Use <ul> quando a ordem não for importante.
- Use <ol> quando a sequência dos itens tiver relevância.
- Use <dl> quando precisar associar termos a descrições.
