Convenções para este treinamento

Linguagens utilizadas

Durante este treinamento, utilizaremos três linguagens principais:

HTML5

Será a base estrutural de nossas páginas, definindo o conteúdo e a organização das informações.

CSS3

Responsável pelo estilo, layout, responsividade e animações, tornando nossas interfaces atrativas e adaptáveis a diferentes dispositivos.

JavaScript

Utilizado para criar interatividade, dinamismo e integrar dados externos às nossas páginas.

Nomes de Diretórios e Arquivos

Para a criação de diretórios e arquivos, não utilizaremos letras maiúsculas, espaços ou caracteres especiais nos nomes. Ex.:

Flávio → flavio

Flávio Ricardo → flavio-ricardo ou flavio_ricardo

Recursos & Ferramentas

Para o desenvolvimento Front-end, utilizaremos os seguintes recursos:

  • Computador: com sistema operacional Windows 10 ou superior, com acesso à internet.
  • Navegador (Browser): Google Chrome, Microsoft Edge ou Mozilla Firefox (preferência para Chrome).
  • Editor de Código: Microsoft Visual Studio Code (preferencial) ou equivalente.
  • Sistema de Versionamento e Gerenciamento de Repositórios: Git e GitHub.
  • Ferramentas de apoio: extensões do VS Code, inspetor de elementos do navegador e serviços online para testes.
