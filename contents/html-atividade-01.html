HTML - Atividade 01

Utilizando a estrutura básica de um arquivo HTML, crie uma nova pasta chamada atividade-01 e nela crie um arquivo chamado index.html que contenha um texto sobre O que é a internet, contendo:

  • Um título principal (<h1>).
  • Texto de parágrafo (<p>), mínimo de 6 parágrafos.
  • O texto de parágrafo deve conter as tags <strong>, <em>, <ins>, <del>, <sub>, <sup>.
  • Inclua pelo menos dois subtítulos no texto.
  • Inclua pelo menos uma quebra de linha no texto.
  • Inclua pelo menos uma linha horizontal no texto.
