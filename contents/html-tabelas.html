Tabelas em HTML

Em HTML, tabelas são usadas para organizar dados em linhas e colunas. Elas são ideais para exibir informações tabulares, como horários, preços, estatísticas ou qualquer dado que precise de comparação organizada. Uma tabela é composta principalmente por três partes: cabeçalho, linhas e células.

Estrutura Básica de uma Tabela (<table>)

A estrutura básica de uma tabela utiliza a tag <table>. Dentro dela, cada linha é definida com <tr> (table row), e cada célula da linha pode ser um <td> (table data). 

<table>
    <tr>
        <td>Linha 1, Coluna 1</td>
        <td>Linha 1, Coluna 2</td>
    </tr>
    <tr>
        <td>Linha 2, Coluna 1</td>
        <td>Linha 2, Coluna 2</td>
    </tr>
</table>
Linha 1, Coluna 1 Linha 1, Coluna 2
Linha 2, Coluna 1 Linha 2, Coluna 2

Por padrão, o HTML não exibe as bordas da tabela, permitindo que o foco dessa estrutura sejam os dados, uma vez que utilizaremos CSS para estilização. Entretanto, apenas como forma de ter um auxílio visual na construção das tabelas, inclua o atributo border="1" na tag <table> como mostrado abaixo:

<table border="1">

Cabeçalho da Tabela (<th>)

Para destacar o título de uma coluna ou linha, utilizamos a tag <th> em vez de <td>. O conteúdo de <th> aparece em negrito e centralizado por padrão. 

<table>
    <tr>
        <th>Produto</th>
        <th>Preço</th>
    </tr>
    <tr>
        <td>Notebook</td>
        <td>R$ 3.500</td>
    </tr>
    <tr>
        <td>Mouse</td>
        <td>R$ 80</td>
    </tr>
</table>
Produto Preço
Notebook R$ 3.500
Mouse R$ 80

Mesclando Células (<rowspan> e <colspan>)

Para unir células horizontalmente, usamos o atributo colspan. Para unir verticalmente, usamos o rowspan. 

<table>
    <tr>
        <th>Nome</th>
        <th colspan="2">Contatos</th>
    </tr>
    <tr>
        <td>Ana</td>
        <td>Email</td>
        <td>Telefone</td>
    </tr>
</table>
Nome Contatos
Ana Email Telefone

Resumo

- Use <table> para criar uma tabela.
- Use <tr> para definir uma linha.
- Use <td> para células de dados.
- Use <th> para cabeçalhos.
- Use colspan e rowspan para unir células.

Estrutura Avançada (<thead>, <tbody> e <tfoot>)

Para organizar melhor a tabela, podemos separar o conteúdo em três partes:

  • <thead> → usado para o cabeçalho da tabela.
  • <tbody> → usado para o conteúdo principal da tabela.
  • <tfoot> → usado para o rodapé, geralmente com totais ou observações.

Essa estrutura é especialmente útil para tabelas grandes, pois ajuda na semântica, acessibilidade e até na impressão de documentos. 

<table>
    <thead>
        <tr>
            <th>Produto</th>
            <th>Quantidade</th>
            <th>Preço</th>
        </tr>
    </thead>

    <tbody>
        <tr>
            <td>Notebook</td>
            <td>2</td>
            <td>R$ 7.000</td>
        </tr>
        <tr>
            <td>Mouse</td>
            <td>3</td>
            <td>R$ 240</td>
        </tr>
    </tbody>

    <tfoot>
        <tr>
            <td colspan="2">Total</td>
            <td>R$ 7.240</td>
        </tr>
    </tfoot>
</table>
Produto Quantidade Preço
Notebook 2 R$ 7.000
Mouse 3 R$ 240
Total R$ 7.240

Considerações finais

- Use <table>, <tr>, <td> e <th> para a estrutura básica.
- Use colspan e rowspan para mesclar células.
- Use <thead>, <tbody> e <tfoot> para tabelas grandes e bem organizadas.
- Evite usar tabelas para layout. Hoje, elas devem ser usadas apenas para dados tabulares.
