Começando com CSS

O CSS (Cascading Style Sheets) é a linguagem usada para definir o estilo de páginas web. Enquanto o HTML cuida da estrutura e do conteúdo, o CSS controla cores, fontes, espaçamentos, tamanhos, bordas e posicionamento.

Por que usar CSS?

Com o CSS podemos separar a estrutura (HTML) da apresentação (estilo). Isso deixa o código mais organizado, facilita a manutenção e dá muito mais controle sobre o visual do site.

Formas de aplicar CSS

Existem três formas principais de aplicar CSS em um documento HTML:

  • Inline: o estilo é aplicado diretamente no elemento HTML.
  • Interno: o estilo é escrito dentro da tag <style> no próprio documento.
  • Externo: o estilo fica em um arquivo separado (.css) e é importado no HTML.

CSS inline

<p style="color: blue; font-size: 20px;">
    Este texto está azul e maior.
</p>

Este texto está azul e maior.

CSS Interno

<head>
    <style>
        p {
            color: red;
            font-size: 18px;
        }
    </style>
</head>

<body>
    <p>Este texto está vermelho.</p>
</body>

Este texto está vermelho.

CSS Externo

<head>
    <link rel="stylesheet" href="style.css">
</head>

<body>
    <p>Texto estilizado com CSS externo.</p>
</body>
 
/* Arquivo style.css */
p {
    color: green;
    font-size: 22px;
}

Texto estilizado com CSS externo.

Resumo

- O CSS define o estilo visual do HTML.
- Podemos aplicar CSS de três formas: inline, interno e externo.
- O uso de arquivos externos é a prática mais comum e recomendada.
- CSS dá vida ao HTML: cores, fontes, espaçamentos, tamanhos e muito mais.
