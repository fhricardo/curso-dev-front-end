Cores em CSS

No CSS podemos definir cores de várias formas diferentes. Isso permite flexibilidade para criar desde estilos simples até combinações muito sofisticadas.

1. Cores por nome

O CSS reconhece mais de 140 nomes de cores. Exemplos: red, blue, green, black, white.

Confira a relação completa de cores com nomes no site do W3 Schools clicando aqui.

p {
    color: red;
    background-color: yellow;
}

Texto vermelho com fundo amarelo.

2. Cores em código hexadecimal

O formato #RRGGBB usa números hexadecimais (0–9 e A–F). Cada par representa a intensidade de vermelho, verde e azul (0 a 255). 

p {
    color: #ff0000;   /* vermelho */
    background-color: #00ff00; /* verde */
}

Texto vermelho no fundo verde.

3. Cores em RGB

O formato rgb(r, g, b) define as intensidades de vermelho, verde e azul (0 a 255). 

p {
    color: rgb(0, 0, 255); /* azul */
    background-color: rgb(200, 200, 200); /* cinza claro */
}

Texto azul com fundo cinza.

4. Cores em RGBA

A versão rgba(r, g, b, a) adiciona o canal de transparência (alpha), que vai de 0 (totalmente transparente) até 1 (totalmente opaco). 

p {
    background-color: rgba(255, 0, 0, 0.5); /* vermelho semi-transparente */
}

Fundo vermelho com transparência.

5. Cores em HSL

O formato hsl(h, s%, l%) define: - hue (matiz): a cor na roda de 0 a 360 (ex: 0 = vermelho, 120 = verde, 240 = azul).
- saturation (saturação): de 0% (cinza) até 100% (cor pura).
- lightness (luminosidade): de 0% (preto) até 100% (branco). 

p {
    color: hsl(200, 100%, 40%); /* azul forte */
    background-color: hsl(50, 100%, 80%); /* amarelo claro */
}

Texto azul sobre fundo amarelo claro.

Resumo

- Podemos usar cores por nome (simples, mas limitado).
- Hexadecimal (#RRGGBB) é um dos formatos mais comuns.
- rgb() e rgba() permitem mais controle e transparência.
- hsl() facilita criar variações de saturação e luminosidade.
