Visual Studio Code

O Visual Studio Code, também conhecido como VS Code, é um dos editores de código mais populares e eficientes do mercado. Desenvolvido pela Microsoft, ele oferece suporte a diversas linguagens de programação, incluindo HTML, CSS, JavaScript, PHP, Python, entre muitas outras. É leve, rápido e extremamente personalizável graças à sua vasta biblioteca de extensões.

Durante este curso, utilizaremos o Visual Studio Code como nosso editor principal, pois ele reúne recursos como realce de sintaxe, auto-completar, integração com Git e depurador, tornando o desenvolvimento muito mais ágil e produtivo.

Baixando o Visual Studio Code

O instalador do Visual Studio Code para Windows pode ser obtido no site oficial: https://code.visualstudio.com/.

Ao acessar o site, clique no botão "Download for Windows". O download do instalador será iniciado automaticamente.

Instalando o Visual Studio Code

Após baixar o instalador, execute o arquivo e siga o assistente de instalação.

Inicialmente, aceite o Acordo de Licença

Em seguida, é muito importante marcar a opção "Add 'Open with Code' action to Windows Explorer file context menu" (Adicionar ação "Abrir com Code" ao menu de contexto do Windows Explorer) tanto para arquivos quanto para pastas.

Essa opção permite que você clique com o botão direito em qualquer arquivo ou pasta no Windows e escolha a opção "Abrir com Code". Isso agiliza muito o trabalho, pois você poderá abrir um projeto ou arquivo específico diretamente no editor sem precisar navegar manualmente até ele.

Em seguida, será exibida uma tela de confirmação das opções selecionadas. Agora, basta clicar em Instalar.

Agora é só esperar o processo de instalação finalizar...

Após o término do processo, é só clicar em Concluir

Primeira execução

Após finalizar a instalação, abra o Visual Studio Code. Na primeira execução, você verá uma tela de boas-vindas com sugestões de temas, atalhos e extensões. Recomendamos que você instale as extensões que iremos indicar ao longo do curso para configurar o ambiente ideal de desenvolvimento.

Com o Visual Studio Code configurado, teremos uma ferramenta poderosa para escrever, organizar e manter nossos códigos com muito mais produtividade.
