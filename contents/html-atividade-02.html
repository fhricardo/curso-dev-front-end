HTML - Atividade 02

Links e Navegação

Crie um pasta chamada atividade-02 e abra com o VS Code.

Nesta pasta, crie três arquivos, com os seguintes nomes:

  • index.html
  • sobre.html
  • contato.html

Em cada um dos arquivos, insira a estrutura básica do HTML, e um menu que permita a navegação entre os arquivos

Para distinguir os arquivos, insira em cada um deles a tag <h1> com os títulos das páginas.
