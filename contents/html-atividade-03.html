HTML - Atividade 03

Trabalhando com Listas

Crie um pasta chamada atividade-03 e dentro dela, crie os seguintes arquivos:

  • index.html
  • como-fazer.html
  • descritivo.html

Página index.html

A página index.html deve conter:

  • um título <h1> com o texto Dez coisas que eu quero fazer
  • Uma estrutura de navegação para as outras páginas.
  • Uma lista não-ordenada com dez coisas que você deseja fazer (vale qualquer coisa: viajar para algum lugar, comprar um carro etc.)

Página como-fazer.html

Na página como-fazer.html você deve inserir uma lista ordenada, descrevendo os passos para realizar uma determinada tarefa. O conteúdo dever ter, no mínimo, 6 passos.

Não se esqueça de incluir um título para esta página e uma estrutura de navegação para as demais páginas.

Página descritivo.html

Na página descritivo.html você deve incluir:

  • Estrutura de navegação para as outras páginas.
  • Título da página.
  • Uma lista de definição para os seguintes termos:
    • HTML
    • CSS
    • JavaScript
