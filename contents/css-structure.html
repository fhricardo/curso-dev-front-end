Estrutura Básica do CSS

O CSS funciona com um conjunto de regras de estilo. Cada regra é formada por um seletor e um bloco de declarações.

Anatomia de uma regra CSS

p {
    color: blue;
    font-size: 20px;
}

- Seletor: p → indica a qual elemento HTML a regra se aplica.
- Propriedade: color, font-size → características que queremos mudar.
- Valor: blue, 20px → o resultado aplicado à propriedade.

Este é um parágrafo azul e maior.

Seletores mais comuns

  • Por tipo: seleciona todos os elementos de uma tag. p { color: red; }
  • Por classe: usa um ponto (.) antes do nome. .destaque { background: yellow; }
  • Por id: usa uma cerquilha (#). #titulo { font-size: 32px; }

Exemplo

<h1 id="titulo">Título da Página</h1>
<p>Um parágrafo simples.</p>
<p class="destaque">Um parágrafo em destaque.</p>

#titulo {
    color: darkblue;
}
p {
    color: gray;
}
.destaque {
    background: yellow;
    font-weight: bold;
}

Título da Página

Um parágrafo simples.

Um parágrafo em destaque.

Boas práticas iniciais

- Prefira usar classes ao invés de ids para estilizar vários elementos.
- Mantenha nomes de classes significativos, como .botao-primario ou .menu.
- Use arquivos externos (.css) para organizar melhor o código.

Resumo

- Uma regra CSS é composta por seletor, propriedades e valores.
- Seletores podem ser: por tipo, por classe ou por id.
- O uso de classes é a forma mais prática e comum no dia a dia.
