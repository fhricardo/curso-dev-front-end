Box Model no CSS

O Box Model, ou Modelo de Caixa, é um dos conceitos fundamentais em CSS. Ele define como os elementos HTML são renderizados na página, determinando seu tamanho, espaçamento interno, borda e distância em relação a outros elementos.

Em resumo: todo elemento HTML é representado como uma caixa.

Estrutura do Box Model

Cada caixa é composta por quatro camadas:

  • Content – Área onde fica o conteúdo (texto, imagem, etc.).
  • Padding – Espaçamento interno, entre o conteúdo e a borda.
  • Border – Contorno do elemento (pode ter espessura, estilo e cor).
  • Margin – Espaçamento externo, que separa o elemento de outros elementos.
.caixa {
    width: 100px;
    height: 100px;
    padding: 20px;
    border: 2px solid black;
    margin: 10px;
}

Box-Sizing

Por padrão, o CSS usa o valor content-box, ou seja, a largura e altura definidas no elemento não incluem padding e borda. Isso pode alterar o tamanho final da caixa.

Exemplo: se um elemento tiver width: 100px, padding: 20px e border: 2px, a largura total será:

100px + 20px + 20px + 2px + 2px = 144px

Para facilitar, podemos usar box-sizing: border-box;. Assim, a largura e altura declaradas já consideram o padding e a borda, tornando o controle do layout mais previsível. 

/* Sem border-box */
.caixa1 {
    width: 100px;
    padding: 20px;
    border: 2px solid black;
}

/* Com border-box */
.caixa2 {
    width: 100px;
    padding: 20px;
    border: 2px solid black;
    box-sizing: border-box;
}

Resumo

- Todo elemento é uma caixa (Box Model).
- O tamanho final é composto por: content + padding + border + margin.
- padding = espaçamento interno.
- border = contorno do elemento.
- margin = espaçamento externo.
- Use box-sizing: border-box; para facilitar o controle de largura e altura.
