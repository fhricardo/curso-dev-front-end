Aula - Propriedade Display no CSS

A propriedade display é uma das mais importantes do CSS, pois define como um elemento HTML será exibido na página. Ela controla o comportamento visual dos elementos e influencia no fluxo do layout.

Principais valores de display

  • block: o elemento ocupa toda a largura disponível e sempre começa em uma nova linha. Exemplos: <div>, <p>, <h1>...<h6>.
  • inline: o elemento ocupa apenas o espaço necessário ao seu conteúdo e não quebra linha. Exemplos: <span>, <a>, .
  • inline-block: semelhante ao inline, mas permite definir largura (width) e altura (height).
  • none: oculta completamente o elemento, removendo-o do fluxo da página.
  • flex: transforma o elemento em um container flexível, usado para criar layouts modernos e responsivos.
  • grid: transforma o elemento em um container de grid, permitindo organizar elementos em linhas e colunas.

Exemplos básicos

/* Elemento em bloco */
div {
    display: block;
}

/* Elemento em linha */
span {
    display: inline;
}

/* Elemento em linha com controle de largura/altura */
button {
    display: inline-block;
    width: 120px;
    height: 40px;
}

Display: none vs visibility: hidden

É comum confundir as duas propriedades:

  • display: none → o elemento desaparece e não ocupa espaço.
  • visibility: hidden → o elemento fica invisível, mas ainda ocupa espaço na página.

Flex e Grid (introdução rápida)

Além dos valores básicos, existem os modelos modernos de layout:

  • display: flex → organiza os itens em linha ou coluna, permitindo alinhamentos avançados.
  • display: grid → organiza os itens em linhas e colunas, muito útil para layouts de páginas complexos.

Esses dois modos de layout (flex e grid) serão aprofundados em aulas futuras, mas é importante já conhecer a existência deles.

Resumo

  • block → ocupa a largura toda e quebra linha.
  • inline → ocupa só o espaço do conteúdo e não quebra linha.
  • inline-block → mistura dos dois: fica em linha, mas aceita largura e altura.
  • none → some do layout.
  • flex e grid → usados para layouts modernos e responsivos.

Uma boa forma de aprender display é brincar trocando o valor nos elementos e observando o efeito na página.
