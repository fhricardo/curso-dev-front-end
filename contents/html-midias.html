Mídias em HTML

Em HTML, mídias como áudio e vídeo são usadas para enriquecer o conteúdo da página, deixando-a mais interativa e envolvente. Para inserir esses elementos, utilizamos as tags <audio> e <video>, que possuem tag de fechamento e podem conter diferentes atributos para controlar sua reprodução.

Áudio em HTML (<audio>)

A forma mais simples de inserir um áudio é utilizando a tag <audio> com o atributo src. Para permitir que o usuário tenha controles (play, pause, volume), utilizamos o atributo controls. 

<audio src="musica.mp3" controls></audio>

Formatos de Áudio e Compatibilidade

Nem todos os navegadores suportam os mesmos formatos. Para garantir maior compatibilidade, podemos fornecer mais de uma opção de arquivo usando <source>. 

<audio controls>
    <source src="musica.mp3" type="audio/mpeg">
    <source src="musica.ogg" type="audio/ogg">
    Seu navegador não suporta áudio.
</audio>

Vídeo em HTML (<video>)

Para inserir vídeos, usamos a tag <video>. Assim como no áudio, o atributo controls é usado para exibir os controles de reprodução. Podemos definir largura (width) e altura (height) ou deixá-los responsivos com CSS. 

<video src="video.mp4" width="400" controls></video>

Formatos de Vídeo e Compatibilidade

Assim como no áudio, o ideal é oferecer múltiplos formatos de vídeo. Os mais comuns são MP4 (H.264), WebM e Ogg. 

<video controls width="400">
    <source src="video.mp4" type="video/mp4">
    <source src="video.webm" type="video/webm">
    <source src="video.ogv" type="video/ogg">
    Seu navegador não suporta vídeo.
</video>

Atributos Importantes

Tanto em <audio> quanto em <video>, temos alguns atributos úteis:

  • controls – exibe os controles de reprodução.
  • autoplay – inicia automaticamente (nem sempre funciona, depende do navegador).
  • loop – faz a mídia reiniciar automaticamente ao terminar.
  • muted – inicia o áudio/vídeo sem som.
  • poster (apenas em vídeo) – define uma imagem de capa antes da reprodução começar.
<video src="trailer.mp4" width="400" controls autoplay muted poster="capa.jpg"></video>

Quando a Mídia Não Carrega

Caso o arquivo não exista ou o navegador não suporte o formato, o conteúdo alternativo dentro da tag será exibido. 

<audio src="nao-existe.mp3" controls>
    Seu navegador não conseguiu carregar o áudio.
</audio>

Resumo

- Use <audio> e <video> para inserir mídias em uma página.
- Sempre utilize controls para permitir interação do usuário.
- Forneça múltiplos formatos para garantir compatibilidade entre navegadores.
- Utilize atributos como autoplay, loop e muted com moderação.
- Para vídeos, use poster como capa inicial.
- Sempre inclua um conteúdo alternativo para navegadores que não suportam a mídia.
