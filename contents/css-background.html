Propriedades de Background

O background é o plano de fundo dos elementos HTML. Com ele podemos aplicar cores, imagens, controlar repetição, posição e até o comportamento do fundo ao rolar a página.

Background Color

p {
    background-color: lightblue; /* cor de fundo */
}

Fundo azul claro aplicado ao parágrafo.

Background Image

div {
    background-image: url('assets/sunflower.webp');
}

Background Repeat

Por padrão, a imagem de fundo se repete. Podemos controlar isso:

div {
    background-image: url('assets/sunflower.webp');
    background-repeat: no-repeat; /* outras opções: repeat-x, repeat-y */
}

repeat-x

repeat-y

Background Position

div {
    background-image: url('assets/sunflower.webp');
    background-repeat: no-repeat;
    background-position: right bottom; /* ou valores em px/% */
}

Background Size

Podemos controlar o tamanho da imagem de fundo:

div {
    background-image: url('assets/flower-field.webp');
    background-size: cover; /* outras opções: contain, valores em px/% */
}

background-size: auto

Mostra a imagem em seu tamanho normal, cortando partes que não podem ser exibidas e repetindo onde for necessário (caso a propriedade background-repeat for definida como repeat)

background-size: contain

Redimensiona automaticamente a imagem para certificar que ela seja exibida em sua totalidade, repetindo caso a propriedade background-repeat esteja definida como repeat.

background-size: cover

Redimensiona a imagem para que ela cubra toda a área onde foi aplicada.

Background Attachment

Controla se o fundo rola junto com a página ou fica fixo:

div {
    background-image: url('assets/flower-field.webp');
    background-attachment: fixed; /* ou scroll (padrão) */
}

Exemplo de background-attachment: fixed

Shorthand: Background

Podemos resumir todas as propriedades em uma única linha:

div {
    background: url('assets/sunflower.webp') 
                no-repeat right top / cover 
                fixed;
}

Resumo

- background-color: cor de fundo.
- background-image: adiciona imagem de fundo.
- background-repeat: controla repetição (repeat, no-repeat, repeat-x, repeat-y).
- background-position: posiciona a imagem (top, center, bottom, valores).
- background-size: ajusta o tamanho da imagem (cover, contain, px/%).
- background-attachment: fixa ou rola o fundo (scroll, fixed).
- background: shorthand para agrupar todas as propriedades.
