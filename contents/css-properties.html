Principais Propriedades do CSS

Agora que já entendemos a estrutura básica do CSS, vamos conhecer algumas das propriedades mais usadas no dia a dia para estilizar textos, cores e espaçamentos.

Cores

p {
    color: red;              /* cor do texto */
    background-color: yellow; /* cor de fundo */
}

Texto vermelho com fundo amarelo.

Fontes e textos

p {
    font-family: Arial, sans-serif; /* tipo da fonte */
    font-size: 20px;                /* tamanho do texto */
    font-weight: bold;              /* espessura: normal, bold, etc */
    text-align: center;             /* alinhamento: left, right, center, justify */
}

Texto centralizado, em Arial, maior e em negrito.

Espaçamento

Com margin controlamos o espaço externo do elemento, e com padding o espaço interno.

p {
    margin: 20px;   /* espaçamento externo */
    padding: 10px;  /* espaçamento interno */
    border: 2px solid black; /* adiciona uma borda */
}

Texto com margem, padding e borda.

Exemplo combinado

<p class="caixa">Exemplo de parágrafo estilizado.</p>
.caixa {
    color: white;
    background-color: darkblue;
    font-size: 18px;
    padding: 15px;
    margin: 20px 0;
    border-radius: 8px; /* cantos arredondados */
}

Exemplo de parágrafo estilizado.

Resumo

- color e background-color: controlam cores de texto e fundo.
- font-family, font-size, font-weight, text-align: controlam fontes e textos.
- margin e padding: controlam os espaçamentos.
- border e border-radius: criam e arredondam bordas.
