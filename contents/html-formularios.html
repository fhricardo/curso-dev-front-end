Formulários em HTML

Formulários em HTML permitem coletar informações dos usuários, como textos, escolhas, arquivos e muito mais. Eles são criados com a tag <form> e podem conter diversos tipos de campos de entrada. Através do formulário, os dados podem ser enviados para processamento em um servidor usando diferentes métodos de envio.

Estrutura Básica do Formulário (<form>)

A forma mais simples de criar um formulário é utilizar a tag <form> com os atributos action (para onde os dados serão enviados) e method (forma de envio: get ou post). 

<form action="processa.php" method="post">
    <label for="nome">Nome:</label>
    <input type="text" id="nome" name="nome">
    <input type="submit" value="Enviar">
</form>

Métodos de Envio (GET e POST)

O atributo method define como os dados do formulário serão enviados:

  • GET: os dados aparecem na URL (bom para buscas e formulários simples).
  • POST: os dados são enviados de forma oculta no corpo da requisição (mais seguro e usado para cadastros, logins etc.).
<!-- Exemplo usando GET -->
<form action="busca.php" method="get">
    <input type="text" name="q" placeholder="Buscar...">
    <input type="submit" value="Pesquisar">
</form>

<!-- Exemplo usando POST -->
<form action="cadastro.php" method="post">
    <input type="text" name="usuario" placeholder="Usuário">
    <input type="password" name="senha" placeholder="Senha">
    <input type="submit" value="Cadastrar">
</form>

Principais Tipos de Campos

Dentro de um formulário podemos ter diversos tipos de campos, cada um com um propósito específico:

  • text – campo de texto curto.
  • password – campo para senhas (oculta os caracteres).
  • email – valida formato de e-mail.
  • number – permite apenas números.
  • date – campo para selecionar datas.
  • radio – seleção única em um grupo de opções.
  • checkbox – seleção múltipla de opções.
  • file – envio de arquivos.
  • textarea – campo de texto longo (parágrafos).
  • select – lista suspensa (dropdown).
<form action="#" method="post">
    Nome: <input type="text" name="nome"><br>

Botões em Formulários

Além de <input type="submit">, podemos usar outros botões:

  • submit – envia o formulário.
  • reset – limpa todos os campos.
  • button – botão genérico (pode ser usado com JavaScript).
<form action="#" method="post">
    <input type="submit" value="Enviar">
    <input type="reset" value="Limpar">
    <button type="button" onclick="alert('Olá!')">Clique Aqui</button>
</form>

Resumo

- Use <form> para criar formulários.
- Defina action (para onde enviar) e method (get ou post).
- Utilize diferentes tipos de campos de entrada (text, password, email, radio, checkbox, file, textarea, select).
- Lembre-se de usar <label> para melhorar a acessibilidade e experiência do usuário.
- Use botões para enviar (submit), limpar (reset) ou acionar ações customizadas (button).
