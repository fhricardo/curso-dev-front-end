HTML - Atividade 05

Trabalhando com Imagens

Crie uma pasta chamada atividade-05 e dentro dela, crie os seguintes arquivos:

  • index.html
  • galeria.html
  • teste.html

Crie, dentro da pasta atividade-05 uma pasta chamada assets, onde devem ser armazenadas as imagens locais.

Página index.html

A página index.html deve conter:

  • Um título <h1> com o texto Exercícios de Imagens em HTML.
  • Uma estrutura de navegação para acessar as outras páginas (galeria.html e teste.html).
  • Um pequeno parágrafo explicando que esta atividade é destinada ao aprendizado da tag <img> e seus principais atributos (src, alt, title, width, height).

Página galeria.html

A página galeria.html deve conter:

  • Um título da página.
  • Uma galeria simples com pelo menos 6 imagens locais (salvas dentro da pasta assets).
  • Cada imagem deve ter um atributo alt descritivo.
  • Ao menos uma imagem deve utilizar o atributo title para exibir um tooltip ao passar o mouse.
  • Pelo menos duas imagens devem ser redimensionadas com width ou height.
  • Estrutura de navegação para as demais páginas.

Página teste.html

A página teste.html deve conter:

  • Um título da página.
  • Um exemplo de imagem carregada a partir de um endereço externo (URL absoluta).
  • Um exemplo de imagem propositalmente com caminho errado, para demonstrar a exibição do atributo alt.
  • Um pequeno texto explicando por que é importante sempre definir o atributo alt.
  • Estrutura de navegação para as demais páginas.

Resumo da Atividade

- A página index.html serve como introdução e hub de navegação.
- A página galeria.html pratica o uso de imagens locais, atributos alt, title e redimensionamento.
- A página teste.html mostra o uso de imagens externas, erros de carregamento e reforça a importância do atributo alt.
