Aula - Sizing & Position no CSS

Nesta aula vamos aprender a controlar o tamanho e o posicionamento dos elementos na página. Esses conceitos são fundamentais para estruturar layouts e deixar o design mais organizado.

Controlando Tamanhos - Sizing

As principais propriedades para definir o tamanho de um elemento são:

  • width: define a largura do elemento.
  • height: define a altura do elemento.
  • max-width: largura máxima que o elemento pode ter.
  • min-width: largura mínima que o elemento deve ter.
  • max-height: altura máxima.
  • min-height: altura mínima.
div {
    width: 300px;
    height: 200px;
    max-width: 100%;
}

Usar max-width com 100% é uma prática comum para tornar o layout mais responsivo, evitando que o elemento ultrapasse a largura da tela.

Box-sizing

O box-sizing define como o navegador calcula o tamanho total do elemento:

  • content-box (padrão): largura/altura aplicam-se apenas ao conteúdo. Padding e borda são adicionados depois.
  • border-box: largura/altura já incluem conteúdo, padding e borda.
div {
    width: 200px;
    padding: 20px;
    border: 5px solid black;
    box-sizing: border-box;
}

É comum aplicar box-sizing: border-box; no projeto inteiro, pois facilita o controle de medidas.

Position

A propriedade position controla como os elementos são posicionados na página.

  • static (padrão): segue o fluxo normal da página.
  • relative: permite mover o elemento em relação à sua posição original (top, right, bottom, left).
  • absolute: posiciona o elemento em relação ao ancestral mais próximo que tenha position diferente de static.
  • fixed: fixa o elemento em relação à janela do navegador, mesmo ao rolar a página.
  • sticky: mistura de relative e fixed; o elemento se move até certo ponto e depois “gruda” na tela.

Exemplo de Position

.container {
    position: relative;
    width: 400px;
    height: 200px;
    background: lightgray;
}

.box {
    position: absolute;
    top: 20px;
    left: 30px;
    background: tomato;
    width: 100px;
    height: 100px;
}

Z-index

Apesar de utilizarmos um padrão bidimensional em nossas páginas da web, onde temos como um padrão apenas largura e altura, conseguimos simular a profundidade em nossos projetos com base na ordem em que exibimos os conteúdos.

Quando elementos se sobrepõem, o z-index define quem fica por cima ou por baixo: um valor maior corresponde a um posicionamento superior - mais próximo do usuário - e um valor menor, um posicionamento mais abaixo.

.box1 {
    background-color: red;
    position: absolute;
    z-index: 1;
}

.box2 {
    background-color: blue;
    position: absolute;
    z-index: 2; /* Fica acima da box1 */
}

Resumo

  • Use width e height para definir tamanhos.
  • min- e max- ajudam a dar limites ao dimensionamento.
  • box-sizing: border-box é recomendado para projetos.
  • position controla como o elemento é posicionado na página.
  • z-index resolve sobreposição de elementos.

Teste diferentes combinações de position e z-index para entender como os elementos interagem no layout.
